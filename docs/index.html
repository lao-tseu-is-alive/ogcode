Two ways to work

Build & Plan

Whether you need to ship fast or think first, ogcode has a mode for your workflow.

⚡

Build Mode

Chat directly with an AI agent that can read, write, edit, and execute code in your project — all from the terminal.

  • ✓ Read & understand your entire codebase
  • ✓ Write, edit, and refactor code in real time
  • ✓ Execute shell commands and scripts

  • ✓ Interactive web UI alongside terminal
🗺️

Plan Mode

Collaboratively plan a feature or refactor, then break it into tasks with auto-generated branches, commits, and pull requests.

  • ✓ Collaborative planning with read-only agent
  • ✓ Structured task breakdown with effort estimates
  • ✓ Kanban board with dependencies
  • ✓ Auto-create git branches per task
  • ✓ Auto-generate pull requests on completion

Plan mode workflow

Think first,
ship after

Describe your goal, refine the plan with AI, then let ogcode break it into tasks — each with its own branch, agent, and PR.

1

Describe

Open a new plan and describe your goal. The planning agent reads your codebase and helps refine the approach.

2

Lock Plan

When ready, lock the plan. The agent generates a structured task breakdown with effort, complexity, and dependencies.

3

Execute Tasks

Each task gets its own git branch and isolated agent session. Watch the agent work in real time.

4

Auto PR

When a task finishes, the agent commits its changes and a pull request is automatically created. Dependencies auto-start.

Features

Everything you need

Built for real-world development — not just demos.

🔒

Zero config

Set an API key and go. Auto-detects providers, models, and your project setup.

🌐

Web UI

Interactive web interface alongside the terminal. Manage chats, plans, and tasks visually.

🔀

Git native

Automatic branch creation, commits, and PR generation. Each task gets an isolated workspace.

🧠

Agentic memory

Long-term memory across sessions via MCP-compatible memory servers. Context that persists.

🐳

Docker ready

Run in a container with your workspace mounted. Perfect for CI or isolated environments.

🔌

Multi-provider

Anthropic, OpenAI, OpenRouter, Ollama — use any provider or mix and match.
